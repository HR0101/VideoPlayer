# VideoPlayer (iOS Client)

Macで稼働する「Mac Video Server」とローカルネットワーク経由で連携し，大容量の動画や写真をシームレスにストリーミング再生・管理するためのiOS専用高機能クライアントアプリケーションです．

直感的なグラスモーフィズム（すりガラス）UIを採用し，ネイティブアプリならではの滑らかなジェスチャー操作と強力なバックグラウンド処理を実現しています．

## ✨ 主な機能

### 🎬 リッチなメディア再生体験
* **高品質ストリーミング**: オリジナル画質に加え，ネットワーク環境に合わせて1080p（軽量）や540p（節約）のプロキシ動画へ瞬時に切り替え可能．
* **ドラッガブル・プレイヤー**: 動画再生中に下にスワイプして閉じる，左右のスワイプで前後のメディアへサクサク移動するなど，モダンなジェスチャー操作に対応．
* **レジューム再生**: 一度閉じた動画を開き直すと，前回視聴していた続きから自動的に再生を開始．
* **写真ビューア**: ピンチイン・ピンチアウトによるズームや，パン操作に対応した高解像度写真ビューア．
* **シェイクでランダム再生**: デバイスを振る（シェイク）ことで，アルバム内の動画をランダムに再生．

### 📥 強力なダウンロード＆バックグラウンド処理
* **カメラロールへの直接保存**: サーバー上の動画や写真を，iPhoneの「写真アプリ」へ直接ダウンロード可能．
* **バックグラウンドダウンロード対応**: アプリを閉じたり別のアプリを開いたりしても，iOSのバックグラウンドセッションを利用してダウンロードを継続．
* **Dynamic Island風 UI**: アプリ起動中は，画面上部にダイナミックアイランド風の美しいプログレスバーが浮かび上がり，進捗をリアルタイムに表示．
* **スマートな通知システム**: バックグラウンドでのダウンロード中も，進捗が5%進むごとに通知センターを静かに上書き更新し，完了時には確実にお知らせ．

### 🗂️ 柔軟なメディア管理と操作
* **メディアのアップロード**: iPhone内の動画や写真を，アプリ内から直接Macサーバーの指定アルバムへアップロード．
* **多彩なソート機能**: 「追加日」「撮影日（メタデータ）」「動画の長さ」のそれぞれ新しい順・古い順で瞬時に並び替え．
* **アルバム管理**: 動画の複数選択，他アルバムへの移動，アルバムからの除外，完全削除機能．
* **再生履歴**: 最近再生した動画を「HISTORY」アルバムとして自動管理．

### ⌨️ キーボードショートカット対応
iPadでの利用時や外部キーボード接続時に，YouTubeライクな操作が可能です．
* `Space` / `K` : 再生・一時停止
* `L` / `→` : 10秒進む
* `J` / `←` : 10秒戻る
* `↑` / `↓` : 音量の調整
* `M` : ミュート
* `F` : フルスクリーン切り替え
* `Shift + ←` / `Shift + →` : 前後の動画へ移動

## 🛠 動作環境

* iOS 15.0 以上
* Xcode 14.0 以上（開発用）
* **必須**: 同一ローカルネットワーク上で稼働する [Mac Video Server]

## 🚀 インストールとセットアップ

1. このリポジトリをクローンし，Xcodeで `VideoPlayer.xcodeproj` を開きます．
2. アプリが写真や動画をカメラロールに保存・アップロードできるよう，`Info.plist` に以下のプライバシー要件が追加されていることを確認してください．
   * `Privacy - Photo Library Usage Description` (メディアのアップロード用)
   * `Privacy - Photo Library Additions Usage Description` (ダウンロードしたメディアの保存用)
3. 実機（iPhone / iPad）をMacに接続し，ビルド＆実行します．

## 📱 使い方

1. **サーバーへの接続**: アプリを起動すると，Bonjour（NetService）機能により同一Wi-Fi内のMac Video Serverが自動的に検出されます．
2. **再生**: サムネイルをタップするとプレイヤーが起動します．画質の変更は右上のメニューから行えます．
3. **ダウンロード**: 動画や写真のサムネイルを長押し（Haptic Touch）し，「保存」を選択すると，カメラロールへのバックグラウンドダウンロードが開始されます．
4. **アップロード**: 右上のクラウドアイコン（`icloud.and.arrow.up`）をタップし，iPhone内のメディアを選択するとサーバーへ転送されます．

## 🏗 使用技術
* **UI**: SwiftUI (Glassmorphism, LazyVGrid)
* **Media**: AVKit, AVFoundation, Photos (PhotoKit)
* **Networking**: URLSession (Background Configuration), NetService (Bonjour)
* **System**: UserNotifications (Progress update logic)
