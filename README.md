# VideoPlayer (iOS閲覧用アプリ)

手持ちのMacを動画配信サーバー（親機）として活用し，家庭内のネットワークを通じて大容量の動画や画像をスマートフォンで直接視聴・管理するための専用アプリです．

独自の操作感や，アプリを閉じた状態でも処理を継続できる仕組みなど，利用者の使い勝手を第一に考えて構築しました．

## 💡 主な機能

### 1. 快適な動画再生体験
* **画質の切り替え**: ネットワークの状態に合わせて，元の画質のほか，動作の軽い「高画質版」や「節約版」へ瞬時に切り替えて再生できます．
* **直感的な操作**: 画面を下に引き下げて閉じたり，左右になぞって前後の動画へ移動したりと，指の動きに合わせた自然な操作が可能です．
* **続きから再生**: 一度閉じた動画でも，前回見ていた場面から自動的に再開されます．
* **写真の閲覧**: 画像の拡大・縮小や，動画と同じ感覚で前後を切り替えることができます．
* **振って選ぶ**: 端末を軽く振る（シェイク）だけで，保存されている動画から一つをランダムに選び出し，再生を開始します．

### 2. 確実な保存と画面外での処理
* **写真アプリへの保存**: サーバー上のデータを，スマートフォンの「写真」の中へ直接保存できます．
* **通信の継続**: アプリを閉じたり，他の作業をしたりしていても，画面外でデータの受け取りを継続します．
* **状況の通知**: 通信の進捗を画面上部にカプセル状の表示でリアルタイムに知らせます．また，完了時には通知センターを通じて確実に報告します．

### 3. 柔軟なデータ管理
* **データの追加**: スマートフォン内の動画や画像を，サーバー側の指定した場所へ直接送ることができます．
* **高度な並び替え**: 「アプリに追加した順」だけでなく，「動画が撮影された日」や「再生時間の長さ」など，目的に合わせた順序に並び替えられます．
* **履歴の管理**: 最近見た動画は「再生履歴」として自動的にまとめられます．

### 4. キーボード操作への対応
iPadや外部キーボードを利用する場合，動画サイトのような操作が可能です．
* `スペース` / `K`: 再生と一時停止
* `L` / `→`: 10秒進む
* `J` / `←`: 10秒戻る
* `↑` / `↓`: 音量の調整
* `M`: 消音（ミュート）の切り替え
* `F`: 全画面表示の切り替え
* `Shift + 矢印`: 前後の動画へ移動

## 🛠 動作に必要な環境

* iOS 15.0 以上
* Xcode 14.0 以上（開発・構築用）
* **必須条件**: 同じネットワーク内で [Mac Video Server (サーバー側アプリ)] が稼働していること

## 🚀 導入手順

1. この保存庫（リポジトリ）を複製し，Xcodeで `VideoPlayer.xcodeproj` を開きます．
2. アプリが写真や動画を取り扱えるよう，設定ファイル（Info.plist）に以下の権限が記述されていることを確認してください．
   * `Privacy - Photo Library Usage Description` (データの追加用)
   * `Privacy - Photo Library Additions Usage Description` (データの保存用)
3. 自分のiPhoneやiPadをMacに接続し，アプリをインストール（実行）します．

## 📱 使用方法

1. **接続**: アプリを起動すると，同じWi-Fi内にあるサーバーを自動で探し出します．
2. **視聴**: 一覧から動画を叩く（タップする）と再生が始まります．
3. **保存**: 一覧の画像を長く押し，「保存」を選ぶと端末への保存が始まります．
4. **追加**: 右上の雲の印を叩き，端末内のデータを選ぶとサーバーへ送られます．
